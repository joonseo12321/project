Javascript를 사용한 Chrome App 만들기 클론코딩
Chrome APP 만들기 클론코딩
background.js 
※배경을 랜덤으로 설정해주는 기능입니다.
clock.js
※ 데이터를 받아와서 현재 시간을 알려주는 기닁입니다.
quotes.js
※명언들을 랜덤으로 한문장씩 "명언","명언을 남긴 사람의 성함" 순으로 출력해주는 기능입니다.
greetings.js
※유저의 정보를 localstorage에 저장합니다.
※
※유저 정보 저장이 안될땐 hidden으로, 저장이 되면 greeting message를 보여주는 기능입니다.
todo.js
※사용자가 원하는 내용을 입력하면 localstorage에 저장됨. 또한 옆에 버튼으로 삭제가 가능함.
※localstorage에 저장된 todo를 버튼으로 각각 지울 수 있습니다.
weather.js
openweathermap의 API를 사용하여 위치를 기반으로 그 위치에 날씨, 기온을 띄워준다.
